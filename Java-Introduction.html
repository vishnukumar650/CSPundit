Introduction to Java


  • Java is a high level Object-Oriented programming language.
  • It was invented in the year 1995 by James Gosling.
  • It was initially known as Oak.
  • It was developed to apply it on consumer electronic devices like TVs, VCRs, Ovens etc
  • Java is known for WORA (Write once, Run anywhere)
